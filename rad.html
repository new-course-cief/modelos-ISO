RAD (Rapid Application Development)

RAD es un enfoque de desarrollo de software que se centra en la entrega rápida de aplicaciones funcionales y en la iteración continua del ciclo de desarrollo. El objetivo es acelerar el proceso de desarrollo para obtener una aplicación funcional en un corto período de tiempo. RAD se basa en la colaboración intensiva entre los desarrolladores, los usuarios finales y otros stakeholders, y en el uso de herramientas y técnicas que permiten un desarrollo rápido, como prototipado rápido, modelado visual y generación de código automatizada. RAD se enfoca en la entrega temprana de software funcional para obtener retroalimentación del usuario y realizar ajustes rápidos en el desarrollo en función de esa retroalimentación.
